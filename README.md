## 📂 Contenuti

Il progetto è suddiviso nei seguenti moduli:

### 📊 PCA & Clustering
Analisi del dataset e applicazione di tecniche di **Principal Component Analysis (PCA)** e clustering per la riduzione della dimensionalità e la caratterizzazione dei dati.

---

### 🌐 Workload Characterization
Analisi del carico di lavoro di un web server Apache:
- Capacity Test  
- Analisi del bottleneck  
- Fairness Index  
- Caratterizzazione workload (HL e LL)  
- Design of Experiment

---

### 📈 Regression Analysis
Applicazione di modelli di regressione e analisi statistica:
- Stima lineare  
- Test di normalità  
- Test di Mann-Kendall  
- Stimatore di Theil-Sen

---

### ⚙️ Benchmark
Misurazioni sperimentali e analisi delle performance su diverse configurazioni hardware e workload.

---

### 🔧 RBD & Fault Tree
Analisi dell’affidabilità dei sistemi:
- Reliability Block Diagram (RBD)  
- Fault Tree Analysis  
- Minimal cut-set  
- Studio della ridondanza e affidabilità 

---

### 📉 Reliability Analysis
Calcolo e modellazione dell’affidabilità dei sistemi complessi utilizzando modelli esponenziali e metriche MTTF.

---

### 🧪 FFDA (Field Failure Data Analysis)
Analisi dei failure su sistemi reali:
- Dataset Mercury e BlueGene/L  
- Coalescence Window  
- Distribuzioni dei tempi tra failure  
- Fitting statistico (esponenziale, Weibull)  
- Test di Kolmogorov-Smirnov 

---

## ⚙️ Tecnologie Utilizzate

- Python (analisi dati e script)
- Apache JMeter (testing workload)
- Apache Web Server
- Tecniche di Data Analysis e Machine Learning 

---

## 🎯 Obiettivi

- Analizzare e modellare workload reali
- Valutare performance e affidabilità di sistemi
